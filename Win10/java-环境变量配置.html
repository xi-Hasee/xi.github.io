Java 环境变量配置

  1. 安装目录结构:

    D:\Java\---jdk
        ---jre

  2. 用户环境变量:

    JAVA_HOME:      D:\Java\jdk`(jdk安装目录)
CLASS_PATH:     .;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;
PATH:           ;%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;
