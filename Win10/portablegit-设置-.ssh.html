PortableGit 设置 .ssh
  1. 打开git-bash.exe-->ssh-keygen -t rsa -C “username@example.com”,然后一路 next.
  2. 在 user 目录下的 .ssh 文件夹中打开id_rsa.pub,把其中的内容填到 github 的 ssh 中.
